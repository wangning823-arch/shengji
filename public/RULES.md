# 打升级（拖拉机/80分）游戏规则

## 一、基本架构

| 项目 | 说明 |
|------|------|
| 人数 | 4人（固定），2v2对家合作 |
| 牌数 | 2副/3副/4副牌模式 |
| 座位 | 东南西北四方，对家为队友（1-3一家，2-4一家） |
| 目标 | 抢分升级，先打过A的一方获胜 |

## 二、牌型与大小规则

### 主副区分

- 每局通过亮主确定主牌花色（黑桃/红桃/方块/梅花），剩余三门为副牌
- **无主模式**：所有花色均为副牌，仅大小王和级牌为主
- **级牌**：当前庄家级别的牌（如打2时，所有2为主牌）

### 主牌大小顺序（以2副牌为例）

```
大王 > 小王 > 主花色级牌 > 其他花色级牌 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3
```

### 牌型分类

| 牌型 | 说明 | 示例 |
|------|------|------|
| 单张 | 任意一张牌 | 红桃A |
| 对子 | 两张同点数同花色 | 两张红桃A |
| 拖拉机 | 相邻点数的连对（主/副均可） | 红桃KK+红桃QQ（主牌拖拉机） |
| 甩牌 | 同时出多张符合规则的牌 | 需确认是否为同花色 |
| 四张/多张 | 3副及以上时的多张相同牌 | 三张红桃A |

### 拖拉机规则

- 必须是同花色、相邻点数、且均为对子/多张
- 级牌参与时，级牌与其他主牌可以组成拖拉机（如主2+主A可连）
- 小王+大王可组成拖拉机
- 副牌拖拉机不能跨越级牌断点（如打2时，副牌A和K不能连，因为级牌2夹在中间）

## 三、亮主与反主规则

### 首次亮主

- 至少 **2张同花色级牌**（如2张红桃2）
- **不能带王**

### 反主规则

反主必须用**王 + 级牌**，王用于"撬"别人的主：

| 别人亮的主 | 你反主需要 |
|-----------|-----------|
| 2张级牌 | 1张王 + 2张其他花色级牌 |
| 3张级牌 | 1张王 + 3张其他花色级牌 |
| 4张级牌 | 1张王 + 4张其他花色级牌 |

- **级牌数必须 >= 别人亮主的级牌数**
- 如果级牌数相同，**王必须更大**

### 王的大小顺序

先比**数量**，再比**大王数量**：

```
1小王 < 1大王 < 2小王 < 2大王 < 3小王 < 3大王 < ...
```

### 无主

- 纯王 >= 2张可亮无主
- 反无主只能用更多或更大的王

### 示例

- 别人用2张红桃2亮主 → 你选1张大王 + 2张梅花2，可以反
- 别人用1张小王 + 2张方块2反的 → 你选1张大王 + 2张黑桃2，可以反
- 别人用1张大王 + 2张方块2反的 → 你选2张大王 + 2张黑桃2，可以反
- 别人用3张级牌亮主 → 你必须王 + >=3张级牌才能反

## 四、出牌与得分规则

### 得分牌

| 牌 | 分值 |
|---|------|
| 5 | 5分 |
| 10 | 10分 |
| K | 10分 |

- 每副牌共100分，多副牌总分 = 100 × 牌副数

### 毙牌规则

- 首家出副牌，若某玩家无此花色，可用主牌毙
- 多家毙牌时，主牌大者获胜

### 甩牌规则

- 首家可甩牌（一次性出多张同花色组合牌）
- 甩牌必须保证无人能大过其中任何一部分，否则判定为"违规甩牌"
- **违规甩牌处理**：系统自动识别违规甩牌，自动将甩牌拆成最小牌组合打出

## 五、升级规则

升级阈值按 **step = 牌副数 × 20** 计算：

| 闲家得分 | 结果 |
|---------|------|
| 0分（大光） | 庄家方升3级 |
| >0 且 < step | 庄家方升2级 |
| ≥ step 且 < step×2 | 庄家方升1级 |
| ≥ step×2 且 < step×3 | 换庄，不升级（庄下） |
| ≥ step×3 | 闲家方上台并升级，升级级数 = floor(得分/step) - 2，上不封顶 |

### 各副数具体分界点

| 牌副数 | step | 分界点 |
|--------|------|--------|
| 2副牌 | 40 | 0 / 40 / 80 / 120 / 160 / 200 |
| 3副牌 | 60 | 0 / 60 / 120 / 180 / 240 / 300 |
| 4副牌 | 80 | 0 / 80 / 160 / 240 / 320 / 400 |

### 过A规则

- 必须打A且获胜（闲家得分未达庄下标准）才算过A成功
- 若庄下则留在A继续打，不退回

## 六、底牌规则

### 底牌数量

| 牌副数 | 底牌数量 |
|--------|----------|
| 2副牌 | 8张 |
| 3副牌 | 6张 |
| 4副牌 | 8张 |

### 拿底牌

- 亮主/反主结束后，**庄家**获得全部底牌，底牌加入庄家手牌
- 庄家需要从手牌中选出与底牌数量相同的牌，扣回作为新的底牌
- 扣底完成后，游戏正式开始

### 底牌得分

- 底牌中可能包含分牌（5=5分，10=10分，K=10分）
- 若**闲家**赢得最后一轮，则获得底牌中的所有分数
- 若**庄家**赢得最后一轮，底牌分数归庄家方（已计入庄家得分）

## 七、扣底规则（抠底翻倍）

- 若闲家最后一轮用**拖拉机**抠底，底分翻倍
- **翻倍倍数 = 抠底拖拉机所包含的牌张数**
- 例如：用3344（4张牌）抠底，底分翻4倍；用334455（6张牌）抠底，底分翻6倍

## 八、荒庄处理

- 发牌至倒数第二张时暂停，系统询问所有玩家是否亮主
- 若有人亮主，正常进入反主阶段
- 若无人亮主，则发最后一张牌，以最后一张牌的花色作为本局主牌
- 无主情况：最后一张牌为级牌或其他特殊处理时，按具体牌面规则判定
